# Contributing

Thanks for your interest in the Banking System Application.

## Getting started

1. Fork and clone the repository.
2. Copy `.env.example` to `.env` and fill in local values.
3. Backend: see `BankingSystemAPI/` (Spring Boot, Maven).
4. Frontend: see `banking-frontend/` (React + Vite).

## Development workflow

- Create a feature branch off `main`: `git checkout -b feature/short-description`.
- Keep commits focused and write clear commit messages.
- Run the backend and frontend locally before opening a pull request.

## Pull requests

- Describe what changed and why.
- Reference any related issue.
- Ensure the build passes and no secrets are committed.

## Reporting issues

Open a GitHub issue with steps to reproduce, expected behavior, and actual behavior.
